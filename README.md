# 📺 PRIME IPTV — Landing Page

Landing page moderna e responsiva desenvolvida para divulgação e venda de serviços IPTV.

O projeto foi criado com foco em **conversão de clientes**, apresentando planos, benefícios e conteúdos de forma visual e organizada, com **animações de rolagem e design moderno inspirado em plataformas de streaming**.

---

# 🚀 Demonstração

O site apresenta:

* Interface moderna estilo streaming
* Navegação suave entre seções
* Animações ao rolar a página
* Layout responsivo
* Integração com WhatsApp para vendas

Estrutura pensada para **landing pages de alto desempenho**.

# 🎬 Animações

As animações da página são controladas pela biblioteca **AOS (Animate On Scroll)**.

Os elementos aparecem conforme o usuário rola a página.

Exemplo:

```
data-aos="fade-up"
data-aos-delay="200"
```

Configuração:

```
AOS.init({
 duration:1000,
 once:false,
 mirror:true
});
```
